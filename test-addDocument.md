## Documento de Teste para AddDocument Tool

Este é um documento de teste para verificar se a ferramenta `addDocument` está funcionando corretamente com a fila de processamento.

### Entidades de Teste
- CappyRAG Framework: Sistema de processamento de documentos
- VS Code: Editor de código 
- LLM: Large Language Model
- DocumentProcessingQueue: Fila de processamento

### Relacionamentos de Teste
- CappyRAG Framework INTEGRATES_WITH VS Code
- CappyRAG Framework USES LLM  
- CappyRAG Framework INCLUDES DocumentProcessingQueue
- DocumentProcessingQueue PROCESSES Documents

### Conceitos Importantes
- Background Processing: Processamento em segundo plano
- Entity Extraction: Extração de entidades
- Relationship Mapping: Mapeamento de relacionamentos
- Knowledge Graph: Grafo de conhecimento
