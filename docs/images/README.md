# Screenshots

This directory contains screenshots and visual assets for the Cappy README.

## Required Images

- `graph-visualization.png` - Interactive knowledge graph interface
- `chat-interface.png` - AI chat sidebar in VS Code
- `workspace-scanner.png` - Workspace scanning progress and results

## Image Guidelines

- **Format**: PNG preferred for screenshots
- **Resolution**: High DPI (2x) for clarity
- **Size**: Optimize for web (< 500KB per image)
- **Content**: Show actual Cappy interface with realistic data
- **Annotations**: Add callouts or highlights where helpful

## Creating Screenshots

1. Use Cappy with a real project
2. Capture key features in action
3. Ensure UI is clean and readable
4. Include relevant context (file names, code snippets)
5. Optimize images before committing
